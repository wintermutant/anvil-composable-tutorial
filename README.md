# Deploying to Anvil Composable with Kubernetes

A complete tutorial for deploying a full-stack application (Frontend, Backend, Database) on Anvil Composable using Kubernetes.

## Overview

This repository contains a step-by-step guide and all necessary configuration files to deploy a three-tier application:

- **Frontend**: Svelte application
- **Backend**: FastAPI (Python)
- **Database**: MongoDB

## What You'll Learn

- Building and pushing Docker images
- Creating Kubernetes Deployments, Services, and Ingress
- Deploying a multi-tier application on Anvil Composable
- Testing and troubleshooting Kubernetes applications
- Best practices for production deployments
